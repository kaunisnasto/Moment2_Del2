Om denna webbplats


Denna webbplats består av uppgift 2.

Webbplatsen visar mina arbeten i en tabell.

Syftet med uppgiften är att koppla en databas till en server i Node.js samt att använda CRUD med REST API.


Val av databasen


Arbeten inhämtas från en SQLite-databas, som lagras internt i applikationen.

Jag använder SQLite, eftersom denna databas ligger internt i applikationen.

Data inhämtas från databasen med hjälp av REST API och implementeras med CRUD (CREATE, READ, UPDATE, DELETE).

SQLite ligger i mappen "database" på en fil. Eftersom denna databas är relativt liten, så kan
informationen sparas internt i applikationen och behöver inte en separat uppkoppling till databasen.

Jag använder verktyget ARC för att visa implementeringen av CRUD med hjälp av REST API.

CRUD och REST API i uppgiften har implementerats med hjälp av fyra metoder GET, POST, PUT och DELETE.
